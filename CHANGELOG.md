# CHANGELOG — UNDERGROUND FACTORY

Формат: [SemVer](https://semver.org/lang/ru/). Каждый пуш в ветку деплоя = запись здесь + бамп версии в `package.json`. Версия и дата сборки выводятся в футере сайта.

## [0.36.0] — 2026-07-10

### Исправлено
- **GPT IMAGE «глючил» — корень найден**: долгие генерации (gpt-image-2 high — 30-60 c) убивал таймаут серверной функции (~10-26 с). Вся генерация переведена на асинхронные job: сервер сразу отдаёт номер задачи, клиент опрашивает статус и забирает готовые байты — теперь любой модели можно думать хоть 4 минуты.
- **Галерея показывала не то**: Img предпочитает оверрайд по seed, а у всех фото-слайдов карточки seed был общий — после «Применить» каждый слайд показывал одно и то же фото. У каждого слайда теперь свой seed; лайтбокс показывает то же, что и шапка; «Применить» больше не плодит дубль в альбоме.

### Изменено
- **Кнопка «Генерировать» убрана**: клик по кнопке нейросети сам запускает генерацию ею; «⚡ ВО ВСЕХ СРАЗУ» — все параллельно. Нажатая кнопка красиво мерцает до завершения, потом снова обычная.
- **Перегенерация заставки — автоматическая через Replicate**: кнопка «🎬 Заставка»/«🔁 Перегенерить» сама генерит видео (kling v3, ~1-5 мин), кнопка мерцает «ГЕНЕРИМ…», результат сразу оживает в карточке (видео-оверрайд браузера); Claude получает заявку с готовым файлом — закоммитить для всех посетителей. Без ключа — откат на старую очередь.
- **Агент арт-директора наполнен скилами фотореализма**: конкретные камеры и оптика, одна световая ситуация, честные несовершенства (пыль, прохожие, неровный асфальт), документальная композиция, зерно и халяция, жёсткие анти-CGI негативы — вплетается в carPrompt и улучшатель. Плюс агент `art-director` в .claude/agents для терминала.
- **Чистка**: галочка «в галерее» и её ключи/стили удалены, мёртвые кнопки (Генерировать/Ещё вариант/Вернуть исходник/БЕСПЛАТНО) и togglePhoto вычищены.

## [0.35.0] — 2026-07-10

### Добавлено
- **Улучшатель кастомных промптов ✨** — иконка в углу каждого поля промпта (арт-редактор, завод рилсов: сценарий и титры): агент переписывает сырой промпт в детальный, замысел пользователя — закон, только обогащение (композиция, оптика, свет, среда). Для тачек вплетает точный кузов поколения; для рилсов сохраняет язык брифа. Результат кладётся обратно в поле — можно править дальше.
- **Банк ракурсов**: тачки больше не выходят одной изометрией — агент каждый раз получает случайный ракурс (низкий фронт, корма, профиль, сверху, через толпу…) с запретом копировать ракурс референса; явный ракурс в промпте пользователя — сильнее банка.

### Исправлено
- **Библиотека «не пополнялась»** (по фидбеку на Ниссан): фото склада — тяжёлые dataURL, и localStorage упирался в квоту ~5МБ — записи молча пропадали. Склад переехал в IndexedDB (без лимита), старые данные мигрируют автоматически.
- **Заставка BMW X5 — от настоящего фото карточки**: новый мост донёс главное фото (серебристый widebody E53 с карбоновым капотом) — заставка и фото в репозитории заменены; чёрный «придуманный» X5 убран.

## [0.33.0] — 2026-07-10

### Добавлено
- **Библиотека тачки — постоянный склад генераций** (полная переработка логики арт-редактора): у каждой тачки свой склад фото. Окно РЕЗУЛЬТАТОВ копит все генерации (одной моделью и «во всех сразу») до ручной очистки: ✓ — одобрить (фото уходит на склад), ✕ — отклонить, клик — предпросмотр для «Применить». На складе у каждого фото две галочки: зелёная — референс генерации, красная — «в альбоме» (фото попадает в карусель карточки после видео); ✕ удаляет навсегда. «Применить» делает фото главным и одобряет его в альбом.
- **Заявка на заставку прикладывает главное фото**: фото кастомных тачек живут только в браузере админа — теперь кнопки «🎬 Заставка»/«🔁 Перегенерить» выгружают главное фото карточки на мост /api/track, и Claude оживляет именно его.
- **Заявки исполнены**: заставка Nissan 370Z перегенерирована от фото карточки; BMW X5 — фото-канон + заставка парой (фото карточки не добралось до заявки — мост появился только сейчас; фото: /media/cars/bmw-x5-mrf0arcb/photo.jpg); фото Lamborghini Urus «капот и лобовое, Альпы» — /media/cars/lamborghini-urus/photo.jpg (вставьте путь в поле «Фото» формы).

### Исправлено
- **GPT IMAGE не работал**: openai/gpt-image-1 на Replicate требует СВОЙ ключ OpenAI (BYOK) — заменён на openai/gpt-image-2, который ходит по обычному биллингу Replicate одним ключом r8_; качество high.
- **Кастомный промпт побеждает референс**: раньше референс главного фото глушил правки («красный карбоновый обвес» игнорировался всеми моделями) — теперь арт-агент обязан прописывать заказанные изменения явно и жирно, а сервер добавляет каждой модели с референсами инструкцию «правки пользователя СИЛЬНЕЕ референса; референс — только для идентичности кузова».

- **Автоглитч**: фирменное «дрожание» надписей вернулось и работает само по себе — hero-заголовок, заголовок каталога, названия тачек.

### Изменено
- **Машинка-курсор**: честная автомобильная модель (едет только туда, куда смотрит нос, к цели — дугой с заносом); когда курсор замер и флажок пойман — дрифтует вокруг него кругами и восьмёрками с дымом; следы покрышек тают быстрее.
- Админка: заголовок больше не мигает, сканирующая линия медленная и ненавязчивая.
- Режим арт-редактора: рамки на редактируемых картинках заметнее («дышащая» подсветка).
