# CHANGELOG — UNDERGROUND FACTORY

Формат: [SemVer](https://semver.org/lang/ru/). Каждый пуш в ветку деплоя = запись здесь + бамп версии в `package.json`. Версия и дата сборки выводятся в футере сайта.

## [0.21.0] — 2026-07-09

### Добавлено
- **Видеоредактор в «Эфире»**: клик по ролику — большой плеер (пауза, покадрово ±1/24 с, луп отрезка), таймлайн с двумя ползунками. Виртуальная обрезка действует сразу на заставку и Видеосалон (бейдж «✂» на карточке), «Обрезать насовсем» — заявка `trim-video` в очередь; «в эфире»/«БРАК» — прямо в модалке.
- **Музыкальный цех — вкладка «Радио»**: загрузка своих mp3 (сразу в эфир локально; ≤4,5 МБ — авто-заявка на прод через новую функцию `/api/track`), прослушивание, обрезка треков (виртуальная + заявка), «первым в эфире», заявка «разложить на стемы». **Мастеринг** через Web Audio: эквалайзер (низ/середина/верх), компрессор, пресеты HiFi/Club/Radio и ручной режим; авто-выравнивание громкости треков (псевдо-LUFS: замер RMS первых ~5 с, поправка −12…+9 дБ на трек).
- **Контент-завод — вкладка «Контент»**: заказ рилса 9:16 кнопками — тачка, обвес (по совместимости), сценарий («до/после», «трансформация в экшне»: гонка/дрифт/погоня, свой промпт), музыка из радио-каталога, глитч-текст (слоганы/описание/свой), CTA, 15/30 с. Живой предпросмотр сценария «хук от кульминации → биты → CTA». Заявка `reel` уходит в очередь — Claude генерирует клипы (Higgsfield 9:16), собирает ffmpeg-ом с музыкой и глитч-титрами и выкладывает на полку готовых рилсов (скачивание для соцсетей). Голос и правила — в новом `BRANDBOOK.md`.
- **ПИТ-БОСС ожил**: на кнопке рации — живой робот из карбона с красным визором: глаз следит за курсором, при быстром движении мыши смешно машет руками-лопатами и трясёт антенной (выключается в calm/на тачах/при reduced-motion).

### Исправлено
- **Бот снова здоровается**: приветствие было одноразовым навсегда (localStorage) — теперь раз за визит: первый раз легенда завода, дальше — случайный из 4 продающих заходов на сленге с кнопками «хиты/скидка».
- Сервер очереди: границы обрезки теперь сохраняются и для `trim-audio` (раньше — только для видео).

Fitment расширен — у каждой из трёх тачек полное меню обвесов.
- **Дрифт-серия** для остальных пяти тачек (AE86, E36, EK9, R34, LADA 2107) — фото в дрифте, ночь/дым/неон. Все 26 картинок сгенерированы через Higgsfield (soul_cinematic + nano_banana_2).

- **Автоглитч**: фирменное «дрожание» надписей вернулось и работает само по себе — hero-заголовок, заголовок каталога, названия тачек.

### Изменено
- **Машинка-курсор**: честная автомобильная модель (едет только туда, куда смотрит нос, к цели — дугой с заносом); когда курсор замер и флажок пойман — дрифтует вокруг него кругами и восьмёрками с дымом; следы покрышек тают быстрее.
- Админка: заголовок больше не мигает, сканирующая линия медленная и ненавязчивая.
- Режим арт-редактора: рамки на редактируемых картинках заметнее («дышащая» подсветка).
