# CHANGELOG — UNDERGROUND FACTORY

Формат: [SemVer](https://semver.org/lang/ru/). Каждый пуш в ветку деплоя = запись здесь + бамп версии в `package.json`. Версия и дата сборки выводятся в футере сайта.

## [0.7.0] — 2026-07-09

### Добавлено
- **Паспорта материалов**: клик по карточке «Три материала» на главной открывает модалку с видео производства (сгенерированы: карбон/автоклав, ручная укладка композита, литьё АБС), глитч-заголовком, историей и техпараметрами. Редактирование — Админка → МАТЕРИАЛЫ.
- **Вкладка «ЦЕХ» в админке**: дерево «тачка → детали → материал», привязка/отвязка деталей от тачки, режим «⚡ Быстрая деталь» (название+материал+цена — остальное автозаполняется), загрузка STL к детали (под точную генерацию по геометрии).
- **Карусель тачек в каталоге**: чипы брендов + лента карточек (крутится колёсиком); один клик — фильтр каталога по тачке, двойной — большая карточка тачки.
- **Живая карточка тачки**: модалка с «оживающим» видео (сгенерировано по фото каждой тачки, image-to-video), фильтром деталей по материалу и кнопкой «КУПИТЬ ВЕСЬ КИТ ▸ $…».
- **Бесплатный генератор в арт-редакторе** (Pollinations, без ключа) — по умолчанию; GPT Image / Nano Banana 2 остаются для работы по ключу.
- Монтаж hero расширен до 13 эпизодов (все тачки каталога), старт со случайного.

### Изменено
- Губа, капот и спойлер перегенерированы в стандартном студийном стиле (по 2 варианта) — рендеры «на белой тачке» из Stitch убраны.
- Окно арт-редактора: жёсткая фиксация, перетаскивание за шапку, поднято поверх бота (кнопки больше не перекрываются).
- Глитч-надписи работают механикой первого варианта (бегущие полосы) со вспышками.

## [0.4.0] — 2026-07-08

### Добавлено
- **Полный сет для трёх тачек** (Silvia S15, Supra A80, RX-7 FD): сгенерированы главные фото (гаражная серия, единый стиль) и «обвесы в воздухе» — студийные рендеры деталей на чёрном фоне с красной подсветкой, по 2 варианта на каждый из 9 товаров. Fitment расширен — у каждой из трёх тачек полное меню обвесов.
- **Дрифт-серия** для остальных пяти тачек (AE86, E36, EK9, R34, LADA 2107) — фото в дрифте, ночь/дым/неон. Все 26 картинок сгенерированы через Higgsfield (soul_cinematic + nano_banana_2).
- **Кодовый замок админки**: ссылка «Админка» видна только при наведении, вход по коду персонала.
- **Автоглитч**: фирменное «дрожание» надписей вернулось и работает само по себе — hero-заголовок, заголовок каталога, названия тачек.

### Изменено
- **Машинка-курсор**: честная автомобильная модель (едет только туда, куда смотрит нос, к цели — дугой с заносом); когда курсор замер и флажок пойман — дрифтует вокруг него кругами и восьмёрками с дымом; следы покрышек тают быстрее.
- Админка: заголовок больше не мигает, сканирующая линия медленная и ненавязчивая.
- Режим арт-редактора: рамки на редактируемых картинках заметнее («дышащая» подсветка).

## [0.3.0] — 2026-07-08

### Добавлено
- **Редизайн по экранам Google Stitch** (все страницы): слоёные поверхности, живое зерно плёнки, терминальные поля с подчёркиванием, кнопки со свечением, HUD-уголки и статус системы на hero, карточки-пластины со сдвигом при наведении и ч/б→цвет на фото, дисплейные цены, hazard-полосы, мигающий «STAFF ONLY» со сканирующей линией в админке, футер с иероглифом-водяным знаком.
- **Картинки из экспорта Stitch в каталоге**: 14 рендеров деталей (Nissan Z, карбон-кевлар) перенесены в `public/img/` и расставлены по товарам.
- **Hero-видео**: сгенерировано нейросетью (Higgsfield, kling3_0_turbo) — ночной дрифт в фирменном стиле, лежит локально в `public/video/`.
- **Версионирование**: версия из `package.json` и дата сборки в футере, ссылка на этот CHANGELOG.

## [0.2.0] — 2026-07-08

### Добавлено
- **АРТ-РЕДАКТОР** — админ-режим правки медиа прямо на сайте: клик по любой картинке/видео-фону открывает панель с промптом; генерация через GPT Image (OpenAI) или Nano Banana 2 (Gemini), автоподбор формата под элемент, стилевой суффикс из дизайн-токенов; замены живут в IndexedDB и снимаются в один клик (Админка → АРТ).
- Кнопка **«Ген ▸ AI»** в формах деталей и тачек; предыдущие фото объекта — референсы для следующей генерации (мульти-референс).
- Замена видео-фона загрузкой файла (mp4/webm).

### Изменено
- **Курсор**: открытое перекрестие в точке клика (центр всегда виден), флажок меньше/полупрозрачный и отходит вбок; кольцо-мишень убрано.
- **Машинка-дрифт**: подъезжает к курсору и останавливается вместо хаотичных «восьмёрок»; руление мягче.
- Единые сиды фото тачек (`car-{id}`) во всех местах сайта.

### Убрано
- `node_modules` и `dist` из git (Netlify собирает из исходников).
