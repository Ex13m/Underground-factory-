## Что это
Магазин тюнинг-обвесов (карбон/композит/АБС) для JDM-тачек. Прототип без
бэкенда: localStorage/IndexedDB-оверлеи поверх сида, REST-фасад `window.UF_API`
(контракт в `API.md`). Дизайн — кибер-индустриальный техвир по экранам Google
Stitch (экспорт распакован в `stitch-export/`, в git не входит).

## Состояние на сегодня (v0.9.x)
- **Весь медиа-контент сгенерирован нейросетями** (Higgsfield: `soul_cinematic`
  фото тачек, `nano_banana_2` предметка деталей, `kling3_0_turbo` видео) и
  лежит локально в каталогизированном хранилище `public/media/`
  (тачка → детали → материал; манифест `public/media/MEDIA.md`).
- **Главная**: монтаж из 13 ночных видео-эпизодов (случайный старт), автоглитч
  заголовков, модалки материалов с видео производства и спеками.
- **Каталог**: самоходная бесконечная карусель тачек (1 клик — фильтр,
  2 — большая карточка с «оживающим» видео, фильтром по материалу и «купить
  весь кит»), мгновенный селектор марка→модель, чипы брендов.
- **Интро**: видео (тачка включает фары, камера наезжает) под графической
  консолью загрузки; уход в сайт через ТВ-помехи. Фавиконка — крутящееся колесо.
- **Админка** (код 4174, ссылка в шапке видна при наведении): вкладки ЦЕХ
  (дерево тачка→детали→материал, ⚡ быстрая деталь, STL), Обвесы, Тачки,
  Заказы, Промо, Материалы (паспорта), Обмен, АРТ (арт-редактор, ключи,
  замены, очередь заявок Higgsfield).
- **Арт-редактор**: клик по любому медиа — плавающее окно с промптом;
  генерация: серверная функция `/api/generate` (Replicate, ключ в Netlify
  env `REPLICATE_API_TOKEN`) → без неё бесплатный Pollinations по прямой
  ссылке; по своим ключам — GPT Image / Nano Banana 2; видео и картинки
  можно грузить файлом; заявки «Заказать ▸ Higgsfield» копятся в очередь
  (выгрузка gen-queue.json → исполняет Claude в терминале).

## Незакрытые нити
1. **Higgsfield Cloud API** (`platform.higgsfield.ai`, auth `Key ID:SECRET`,
   `/v1/text2image/soul`, поллинг `/requests/{id}/status`) — пользователь любит
   генерации Higgsfield; при получении ключей с cloud.higgsfield.ai переписать
   `netlify/functions/uf-generate.mts` на Soul (ключи в env `HF_API_KEY`/`HF_API_SECRET`).
2. **Спецролик для интро**: сгенерировать «фары строго в объектив»
   (kling3_0_turbo) → положить `/media/intro/ignition.mp4` и поменять
   константу INTRO_VIDEO в `src/fx/BootIntro.tsx` (сейчас там live-ролик S15).
3. Заявка пользователя на LADA в очереди генерации (браузерный localStorage) —
   попросить `gen-queue.json` или промпт словами.
4. Репозиторий тяжёлый от видео (~190 МБ) — при жалобах на clone/deploy
   предложить Git LFS или внешний хостинг медиа.
