# 🏁 HANDOFF — передача проекта в Claude Code Terminal

Дата передачи: 2026-07-08. Ветка: `claude/car-tuning-shop-z39a5s` (вся работа здесь, main пуст).

## Что это за проект
**UNDERGROUND FACTORY** — ультрамодный магазин тюнинг-обвесов из карбона / композита / АБС
для JDM-тачек (Silvia S15, Supra A80, RX-7, AE86, E36, Civic EK9, R34, LADA 2107).
Прототип: полный UX без реального бэкенда (localStorage + адаптеры под будущий Stripe/OAuth).
Собран по методике spec-pilot: интервью → утверждённая спека → саб-агенты → verify.

## Хронология решений (пересказ диалога)
1. Заказчик: «ультрамодный магазин обвесов, не клише» + референсы с Pinterest
   (кибер-техвир: красный/чёрный/белый, японские глифы, декали, HUD).
2. Интервью: оплата — заглушка Stripe (реальный подключается позже), вход — демо-OAuth
   (Google/Apple/GitHub), каталог — админ-панель загрузки с фронта, язык — RU+EN.
3. Спека v2+v3 утверждена. Ключевая правка: **ориентация car-first** — сначала выбираешь
   тачку (марка → модель), под неё подбираются киты.
4. Сборка 5 параллельными саб-агентами (витрина / покупательский контур / админка+API /
   курсор-FX / бот+подсказки) на общем каркасе.
5. **Пивот материалов**: изначально была шутка «обвесы из картона» — заказчик решил:
   материалы реальные, выбор при добавлении детали: **карбон / композит / АБС**.
   Редкости → грейды материалов, «дождевой рейтинг» → термостойкость °C,
   бот «Дядя Картон» → «ПИТ-БОСС», промокоды CARBON5/CARBON10/PITSTOP15.
6. Прогон Playwright по чек-листу спеки: **14/14 зелёные** (сквозная покупка, промокод
   через UF_API, добавление товара через API, RU↔EN, FX, бот).
7. Деплой: Netlify, автодеплой из этой ветки — https://incandescent-paletas-b62698.netlify.app
8. Заказчик сгенерировал редизайн в **Google Stitch** по брифу `STITCH-BRIEF.md`.
   Из Stitch получены: `design_tokens_v1.json` (совпали с нашей системой ~1:1),
   `technical_spec_v1.0.md`, `final_project_report_v1.1.md`. Уже интегрировано из этого:
   новые токены (surface-dim/bright, glow-red, success), текстура carbon-fibre,
   интро **SYSTEM BOOT** (сборка машины + HUD-лог, `src/fx/BootIntro.tsx`).

## ⚠️ ГДЕ ОСТАНОВИЛИСЬ — ТЕКУЩАЯ ЗАДАЧА
Полный экспорт дизайна из Stitch — файл **`stitch_single_file_data_hub.zip` (42 МБ)** —
не удалось передать в облачную сессию (лимиты загрузки чата/GitHub-web, сетевая политика
песочницы блокировала Google Drive). Файл лежит:
- у заказчика локально;
- на его Google Drive в папке `111`.

**Задача для локального Claude Code:**
1. Попросить пользователя положить `stitch_single_file_data_hub.zip` в корень проекта
   (или указать путь), распаковать в `stitch-export/` (в .gitignore добавить, если тяжёлый).
2. Открыть экраны Stitch (скорее всего один самодостаточный HTML с вшитыми ассетами)
   и **сравнить два дизайна**: текущий сайт (`npm run dev`) vs экраны Stitch.
   Важно: заказчику **нравится и первоначальный дизайн** — не переписывать вслепую,
   а сделать сравнение (скриншоты бок о бок по страницам: главная, каталог, товар,
   корзина, кабинет) и показать различия с рекомендацией: что взять из Stitch,
   что оставить как есть. Решение — за заказчиком.
3. После решения — перенести выбранные элементы (менять `src/styles/*` и вёрстку страниц;
   логика/сторы/API не трогать), прогнать проверку из CLAUDE.md, закоммитить и запушить —
   Netlify передеплоит сам.
